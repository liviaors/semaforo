Introdução

O Arduino é uma plataforma que possibilita o desenvolvimento de projetos eletrônicos. Ele é composto por hardware e software. Seu principal componente é o microcontrolador, que processa programas e gerencia as entradas e saídas.

Objetivos

O objetivo da aula prática foi observar como funciona todo o processo prático da criação de um objeto em Arduino anteriormente apresentado de forma teórica nas aulas.

Materiais Utilizados

  • Arduino Uno R3
  • LED Verde
  • LED Amarelo
  • LED Vermelho
  • Resistor 150 Ω
  • Placa de Ensaio Pequena
  • Fios jumpers
  • Cabo USB

Código Fonte


// C++ code
void setup()
{
  pinMode(LED_BUILTIN, OUTPUT);
  pinMode(7,OUTPUT);
  pinMode(4,OUTPUT);
  pinMode(2,OUTPUT);
}

void loop()
{
  digitalWrite(7, HIGH);
  delay(2000);
  digitalWrite(7, LOW);
  digitalWrite(4, HIGH);
  delay(2000);
  digitalWrite(4, LOW);
  digitalWrite(2, HIGH);
  delay(2000);
  digitalWrite(2, LOW);
}

Procedimentos

Montagem do Circuito

Organização dos Componentes:

  1. Coloque o Arduino Uno em uma superfície plana e estável.
  2. Separe o protoboard, LEDs, resistores, e fios jumpers.

Conexão dos LEDs:

  1. LED Vermelho: Conecte o terminal positivo ao pino digital 2 do Arduino.
  2. LED Amarelo: Conecte o terminal positivo ao pino digital 4 do Arduino.
  3. LED Verde: Conecte o terminal positivo ao pino digital 7 do Arduino.

Instalação dos Resistores:

Conecte um resistor de 150Ω ao terminal positivo de cada LED.

Conexão ao GND:

Conecte o barramento de terra do protoboard ao pino GND do Arduino Uno usando um fio jumper.

Verificação da Montagem:

Verifique todas as conexões para garantir que os LEDs estejam corretamente conectados.

Configuração do Software

Instale a IDE do Arduino e conecte o Arduino ao computador.
